# Exercício 06: Remoção de Dados

## Objetivo
Aprender a remover registros de uma tabela usando Go.

## Tarefas
1. Escreva um programa Go que:
   - Solicite ao usuário o ID de um usuário.
   - Remova o usuário correspondente da tabela.
   - Exiba uma mensagem de sucesso ou erro.

## Dicas
- Use `db.Exec` para executar comandos de remoção.

---

**Exemplo de saída esperada:**

```
Digite o ID do usuário para remover: 2
Usuário removido com sucesso!
```
